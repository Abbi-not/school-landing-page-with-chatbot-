# School Landing & Application Page

A modern school landing page with an **admin panel** and a **chatbot** that supports **Amharic and English**.  
Includes application forms, interactive chat, and real-time admin management.



# 1. Clone the repo:
git clone https://github.com/Abbi-not/school-landing-page-with-chatbot-.git

cd school-landing-page-with-chatbot-

# 2. Install dependencies:

npm install
cd backend

npm install

# 3. Add `.env` in `backend` with your OpenAI key:

OPENAI_API_KEY=your_key_here

PORT=5000

# 4. Start backend:

cd backend

npm run dev

# 5. Start frontend:

cd .. 

npm run dev
